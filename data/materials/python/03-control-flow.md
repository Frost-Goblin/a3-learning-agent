# 分支与循环

## 学习目标

- 能根据条件控制程序执行路径
- 能用循环处理重复任务
- 能识别常见循环错误并调试

## 分支结构

### 单分支

```python
score = int(input("请输入成绩："))
if score >= 60:
    print("及格")
```

### 双分支

```python
if score >= 60:
    print("及格")
else:
    print("不及格")
```

### 多分支

```python
if score >= 90:
    level = "A"
elif score >= 80:
    level = "B"
elif score >= 70:
    level = "C"
elif score >= 60:
    level = "D"
else:
    level = "E"
```

## 循环结构

### `for` 循环

适合遍历序列或已知次数的重复任务。

```python
for i in range(1, 6):
    print(i)
```

### `while` 循环

适合在条件满足时持续执行。

```python
count = 1
while count <= 5:
    print(count)
    count += 1
```

## 流程控制语句

- `break`：提前结束循环
- `continue`：跳过本次循环余下部分
- `pass`：占位，不执行任何动作

## 易错点

### 1. 忘记更新循环变量

```python
count = 1
while count <= 5:
    print(count)
```

这会导致死循环，因为 `count` 没有递增。

### 2. 条件顺序写错

多分支中，条件要从严格到宽松排列，否则后面的分支可能永远不会执行。

### 3. 循环边界错误

- `range(5)` 生成 `0,1,2,3,4`
- `range(1, 5)` 生成 `1,2,3,4`

### 4. 缩进错误

Python 用缩进表示代码块，条件与循环体必须缩进一致。
