# Python 课程项目模板

## 菜单类项目

### 适合主题
- 学生成绩管理
- 通讯录管理
- 图书借阅记录

### 推荐结构
1. 显示菜单
2. 获取用户输入
3. 调用对应功能
4. 返回主菜单或退出

### 适合训练
- 条件分支
- 循环控制
- 函数拆分
- 列表和字典结合使用

## 统计分析类项目

### 适合主题
- 成绩统计
- 消费记录分析
- 单词频率统计

### 推荐结构
1. 准备数据
2. 遍历数据
3. 统计结果
4. 格式化输出

### 适合训练
- 遍历
- 筛选
- 统计
- 结果整理

## 文件处理类项目

### 适合主题
- 成绩文件读写
- 日志记录
- 文本处理工具

### 推荐结构
1. 读取文件
2. 解析内容
3. 处理数据
4. 写回文件或展示结果

### 适合训练
- 文件读写
- 编码处理
- 异常捕获
- 持久化思维

## 项目拆解通用步骤

1. 明确输入和输出。
2. 确定单条数据和多条数据的表示方式。
3. 拆分核心功能。
4. 补齐异常和边界处理。
5. 准备测试样例。
