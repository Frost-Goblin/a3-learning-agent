# Python 课程知识库

这是一套为 A3 项目准备的首批课程资料，面向 `Python 程序设计` 课程。

设计原则：

- 只保留适合检索和生成的结构化内容
- 以教学大纲、章节讲义、实验任务、综合案例为主
- 每份资料都尽量包含知识点、常见错误、训练任务和交付要求

资料清单：

1. `01-course-outline.md`：课程大纲与学习目标
2. `02-python-basics.md`：变量、数据类型、输入输出
3. `03-control-flow.md`：分支、循环、流程控制
4. `04-functions-modules.md`：函数、参数、模块
5. `05-data-structures-strings.md`：字符串、列表、字典、集合
6. `06-files-exceptions.md`：文件处理与异常调试
7. `07-oop.md`：面向对象基础
8. `08-practice-labs.md`：实验任务与分层训练
9. `09-project-score-manager.md`：综合项目案例

适用方式：

- 作为课程知识库原始资料
- 作为后续 `materials upload / reindex` 功能的默认演示数据
- 作为生成讲解文档、题库摘要、答疑脚本时的检索来源
