# 字符串与常用数据结构

## 学习目标

- 掌握字符串、列表、字典、集合的基本操作
- 能根据问题选择合适的数据结构
- 能完成简单统计、查找和整理任务

## 字符串

字符串用于表示文本数据。

常见操作：

- 索引与切片
- 拼接
- 查找
- 替换
- 分割

示例：

```python
text = "python programming"
print(text[0])
print(text[:6])
print(text.replace("python", "Python"))
```

## 列表

列表适合保存有顺序、可修改的一组数据。

常见操作：

- `append()`
- `insert()`
- `remove()`
- `pop()`
- 遍历

```python
scores = [85, 92, 78]
scores.append(88)
for score in scores:
    print(score)
```

## 字典

字典适合保存“键值对应”关系。

```python
student = {
    "name": "Alice",
    "age": 19,
    "score": 91
}
print(student["name"])
```

典型场景：

- 学生信息
- 商品信息
- 统计词频

## 集合

集合适合去重和成员判断。

```python
nums = {1, 2, 2, 3}
print(nums)
```

## 选择建议

- 单个文本：字符串
- 有顺序的一组值：列表
- 属性描述对象：字典
- 去重或集合关系：集合

## 易错点

### 1. 列表索引越界

### 2. 把字符串当列表修改

字符串不可变，不能直接通过索引修改字符。

### 3. 字典访问不存在的键

建议用 `get()` 降低报错概率。

### 4. 忘记列表和字典可以嵌套

很多真实数据都是“列表里放字典”。
