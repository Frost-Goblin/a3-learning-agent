# Python 基础语法与数据类型

## 学习目标

- 理解变量的含义和赋值规则
- 掌握常见数据类型及其使用场景
- 能完成输入、输出和简单计算

## 核心知识点

### 1. 变量

变量是用于保存数据的名字。Python 中变量不需要先声明类型，直接赋值即可。

示例：

```python
name = "Alice"
age = 19
score = 92.5
passed = True
```

变量命名要求：

- 只能由字母、数字、下划线组成
- 不能以数字开头
- 不能使用关键字
- 建议使用有语义的英文名

### 2. 常见数据类型

- `int`：整数
- `float`：浮点数
- `str`：字符串
- `bool`：布尔值

类型查看：

```python
value = 18
print(type(value))
```

### 3. 输入与输出

输入通常通过 `input()` 获取，返回值默认是字符串。

```python
name = input("请输入姓名：")
age = int(input("请输入年龄："))
print("姓名：", name)
print("年龄：", age)
```

### 4. 运算符

- 算术运算：`+ - * / // % **`
- 比较运算：`> < >= <= == !=`
- 逻辑运算：`and or not`

## 易错点

### 输入值类型错误

`input()` 返回字符串，直接参与数值计算会报错或产生错误结果。

错误示例：

```python
a = input("请输入第一个数：")
b = input("请输入第二个数：")
print(a + b)
```

如果输入 `2` 和 `3`，输出是 `23` 而不是 `5`。

正确写法：

```python
a = int(input("请输入第一个数："))
b = int(input("请输入第二个数："))
print(a + b)
```

### 赋值与比较混淆

- `=` 是赋值
- `==` 是比较

### 字符串与数字直接拼接

错误示例：

```python
age = 18
print("年龄是：" + age)
```

应改为：

```python
age = 18
print("年龄是：" + str(age))
```

## 教学建议

- 初学者先掌握“输入 -> 处理 -> 输出”的三步结构
- 每讲一个类型，配一个生活化例子
- 多做控制台小程序，减少抽象感

## 基础练习

1. 输入姓名和学号，输出格式化的学生信息。
2. 输入长和宽，计算矩形面积与周长。
3. 输入摄氏温度，转换为华氏温度。
4. 输入总分和科目数，输出平均分。

## 提升练习

1. 编写程序，输入半径，输出圆面积和圆周长。
2. 输入两个整数，输出和、差、积、商、整除结果和余数。
3. 输入商品单价和数量，输出总价并保留两位小数。
