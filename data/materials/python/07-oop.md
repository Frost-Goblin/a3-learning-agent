# 面向对象基础

## 学习目标

- 理解类和对象的关系
- 会定义简单类并创建对象
- 能用类描述现实中的实体

## 基本概念

- 类：对象的模板
- 对象：类的实例
- 属性：对象保存的数据
- 方法：对象执行的行为

## 示例

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def show(self):
        print(f"姓名：{self.name}，成绩：{self.score}")
```

创建对象：

```python
s1 = Student("Alice", 90)
s1.show()
```

## 为什么要学面向对象

- 便于描述复杂实体
- 便于组织相关数据和行为
- 更适合做稍大的项目

## 适合教学的对象建模案例

- 学生类
- 课程类
- 图书类
- 订单类

## 易错点

### 1. 忘记写 `self`

实例方法的第一个参数必须是 `self`。

### 2. 混淆类属性和实例属性

### 3. 只会定义类，不会实例化使用

### 4. 一上来就讲继承，超出初学者承受范围

首轮教学应先把“封装对象”讲清楚，再考虑继承、多态。

## 与前面知识的连接

- 函数用于封装动作
- 类用于封装“数据 + 动作”
- 项目越复杂，越需要类和模块配合
