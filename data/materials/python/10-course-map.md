# Python 课程地图

## 课程目标

1. 建立基本编程思维，能把一个任务拆成输入、处理、输出三个部分。
2. 掌握 Python 基本语法、条件分支、循环、函数、常用数据结构。
3. 能完成基础实验、小型作业和一个综合课程项目。
4. 具备初步调试能力，遇到报错时能够定位问题，而不是只会重启程序。

## 课程模块

### 模块 1：开发环境与语法入门
- 代码如何运行
- 缩进为什么会影响结果
- 注释、命名与基础规范

### 模块 2：变量、类型、表达式
- 常见数据类型
- 表达式求值
- 输入为什么常需要类型转换

### 模块 3：条件分支与循环
- 程序怎样做判断
- 程序怎样重复执行任务
- 边界条件如何处理

### 模块 4：函数与模块化
- 为什么不能把所有代码都堆在主流程里
- 如何拆分函数职责
- 模块的复用意义

### 模块 5：字符串、列表、字典、集合
- 一批数据该如何存
- 一条记录该如何表达
- 怎样遍历和统计数据

### 模块 6：文件处理与异常
- 如何保存程序结果
- 文件读写的基本流程
- 报错后如何处理

### 模块 7：面向对象基础
- 什么情况下需要类
- 类和函数的分工是什么
- 如何用对象表示实体

### 模块 8：课程综合项目
- 如何把零散知识点组织成完整程序
- 如何从需求出发拆解功能
- 如何完成一个闭环项目

## 学习递进关系

1. 先掌握变量和流程控制，再进入函数拆分。
2. 先能操作列表和字典，再做文件读写和统计任务。
3. 先完成菜单类小程序，再过渡到综合项目。

## 检索提示

- 学生说“不会开始写”时，优先参考课程地图和项目模板。
- 学生说“总报错”时，优先参考易错点与调试资料。
- 学生说“快考试了”时，优先参考复习提纲和考核标准。
