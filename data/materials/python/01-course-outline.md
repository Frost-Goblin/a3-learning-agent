# Python 程序设计课程大纲

## 课程定位

本课程面向初学编程的本科生和高职学生，目标是让学生建立完整的程序设计基础，并能够独立完成中小型 Python 实践任务。

## 课程目标

1. 理解 Python 基本语法、数据类型和控制结构。
2. 能够使用函数和模块组织代码，而不是只写零散脚本。
3. 掌握常见数据结构与文件读写方法。
4. 初步理解面向对象思想，能设计简单类。
5. 能结合调试、异常处理和测试意识，完成较规范的小项目。

## 核心能力

- 程序阅读能力
- 问题拆解能力
- 基础编码能力
- 调试与纠错能力
- 结构化表达能力

## 章节安排

### 第一章 Python 入门与开发环境

- Python 解释器与脚本执行
- IDE 与命令行基础
- 注释、缩进、命名规范

### 第二章 变量、数据类型与表达式

- 变量定义与赋值
- 数值、字符串、布尔类型
- 运算符与表达式求值
- 类型转换

### 第三章 分支与循环

- `if / elif / else`
- `for` 与 `while`
- `break`、`continue`、`pass`
- 循环嵌套与流程设计

### 第四章 函数与模块

- 函数定义与调用
- 参数与返回值
- 局部变量与作用域
- 模块导入与复用

### 第五章 常用数据结构

- 字符串处理
- 列表操作
- 元组基础
- 字典与集合

### 第六章 文件处理与异常

- 文本文件读写
- 路径与编码
- 常见异常类型
- `try / except / finally`

### 第七章 面向对象基础

- 类与对象
- 属性与方法
- 构造方法
- 封装思想

### 第八章 综合实践

- 小型项目分析
- 模块拆分
- 数据存储
- 结果展示与复盘

## 学习难点

- 能写单条语句，但不会把需求拆成函数
- 能看懂示例，但不能迁移到新题目
- 循环和条件组合时容易漏分支
- 字典、列表、字符串混合使用时容易混淆
- 调试时只看结果，不追踪变量变化

## 建议资源类型

- 分层讲解文档
- 关键知识点图谱
- 基础练习题与错题总结
- 代码实操案例
- 综合项目任务书
